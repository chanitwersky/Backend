

פלטפורמת למידה מבוססת AI - מיני LMS
פלטפורמת Full-stack המאפשרת למשתמשים ללמוד נושאים חדשים באמצעות שיעורים המיוצרים על ידי בינה מלאכותית (AI). המערכת מאפשרת בחירת קטגוריות, הזנת הנחיות אישיות ושמירה על היסטוריית למידה מלאה.

 סקירה כללית
הפרויקט נבנה בדגש על ארכיטקטורה מודולרית, קוד נקי וחווית משתמש חלקה.

Frontend: Angular

Backend: Node.js (Express)

Database: MongoDB

AI Integration: OpenAI API

גישה פיתוחית וארכיטקטורה
צד שרת (Node.js)
השרת מבוסס על מבנה שכבות (Layered Architecture) המפריד בין תחומי האחריות השונים:

Models: הגדרת מבנה הנתונים והקשרים (Schemas).

Repositories: שכבת הגישה לנתונים (Data Access Layer). שכבה זו אחראית באופן בלעדי על ביצוע השאילתות מול בסיס הנתונים.

Controllers: ניהול הבקשות והתגובות (HTTP Requests/Responses).

Services: לוגיקה עסקית ותקשורת מול ה-API של OpenAI.

Routes: הגדרת נתיבי ה-API עבור משתמשים, קטגוריות והיסטוריה.



צד לקוח (Angular)
ממשק המשתמש נבנה כאפליקציית SPA (Single Page Application) מודרנית:

Components: חלוקה לרכיבים לפי פונקציונליות (ניהול , רישום, היסטוריה).

Services: ניהול ריכוזי של קריאות ה-HTTP וניהול נתונים.

UI/UX: שימוש בעיצוב רספונסיבי המותאם ללמידה נוחה.
